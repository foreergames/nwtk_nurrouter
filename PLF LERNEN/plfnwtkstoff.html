Basic Rooter Commands:

User EXEC mode | exit:
Router>
Router> ?
Router> enable (oder en)

---------------------------------------------------------

Privileged EXEC mode | disable, exit:
Router#
Router# ?

---------------------------------------------------------

Global Config mode | exit, end, Ctrl+C, Ctrl+Z:
Router# configure terminal
Router(config)#
Router(config)# ?

---------------------------------------------------------

Line configuration mode | exit, end, Ctrl+C, Ctrl+Z:
Router(config)# line vty 0 15
Router(config)# line console 0
Router(config-line)#
Router(config-line)# ?

---------------------------------------------------------

Interface configuration mode | exit, end, Ctrl+C, Ctrl+z:
Router(config)# interface gigabitEthernet 0/0/0
Router(config-if)#
Router(config-if)# ?
